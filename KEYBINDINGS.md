# tmx 快捷键速查表

## TUI 界面快捷键

| 按键 | 功能 | 说明 |
|------|------|------|
| `↑` / `↓` | 上下导航 | 在会话列表中移动 |
| `k` / `j` | 上下导航 | Vim 风格的导航 |
| `Enter` | 进入会话 | 连接到选中的会话 |
| `n` | 新建会话 | 创建新的 tmux 会话（会提示输入名称） |
| `d` | 断开会话 | 分离选中的会话（detach） |
| `x` | 删除会话 | 永久删除选中的会话 |
| `q` | 退出管理器 | 关闭 TUI |
| `Esc` | 退出管理器 | 关闭 TUI 或取消输入 |

## 新建会话时的快捷键

| 按键 | 功能 |
|------|------|
| `Enter` | 确认创建 |
| `Esc` | 取消创建 |
| `Backspace` | 删除字符 |

## 命令行参数

| 参数 | 功能 |
|------|------|
| `tmx` | 打开 TUI 管理器 |
| `tmx -n <name>` | 快速新建会话 |
| `tmx -a <name>` | 快速连接到会话 |
| `tmx --install` | 安装 tmux 配置 |
| `tmx --uninstall` | 卸载 tmux 配置 |
| `tmx -h` | 显示帮助 |
| `tmx -v` | 显示版本 |

## tmux 快捷键（安装配置后）

| 按键 | 功能 |
|------|------|
| `Ctrl+b t` | 打开会话管理器 |

## 快捷键设计原则

1. **单字母操作** - 所有主要操作都只需要一个字母键
2. **直观易懂** - `n` = new, `d` = detach, `x` = delete (类似 vim)
3. **vim 兼容** - 支持 `j/k` 上下导航
4. **Esc 取消** - 符合通用操作习惯
5. **Enter 确认** - 符合通用操作习惯

## 界面提示

TUI 界面底部会永久显示快捷键提示：
```
[Enter]进入 [d]断开 [n]新建 [x]删除 [q]退出
```

**不需要记忆任何快捷键！**
